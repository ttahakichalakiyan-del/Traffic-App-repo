# Staff App Quick Start

## Warden/Inspector ke liye Asaan Guide

---

### App Install Hone Ke Baad:

1. **Badge ID** aur **PIN** likhein
2. **"Login"** dabayein
3. Pehli baar: 4 setup steps complete karein (sab ✓ green hone chahiye)
4. Home screen par aayein
5. Duty shuru karne ke liye **BADA GREEN BUTTON** dabayein
6. Duty khatam hone par wahi button dobara dabayein (red ho jayega)

---

### Zaroori Baatein:

| Bat | Detail |
|-----|--------|
| 🔋 Battery | 20% se kam ho toh **zaroor charge karein** |
| 📶 Internet | Off ho toh locations **save rehti hain**, internet aane par apne aap bhej deti hain |
| 🔒 Phone Lock | GPS band **NAHI** hoti (agar setup sahi kiya ho) |
| 👁️ DSP Monitoring | DSP saab aapko map par dekh sakte hain **sirf duty hours mein** |

---

### Agar Masla Aaye:

**GPS nahi mil raha?**
→ Bahar jayein, app restart karein

**Login nahi ho raha?**
→ Badge ID mein zeros check karein
→ PIN theek se likha hai? (4 numbers)
→ Admin se rabta karein

**App slow hai?**
→ Phone restart karein
→ Dusri apps band karein
